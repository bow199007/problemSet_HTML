input_password

密码输入区域type="text"转换为type="password"

1、这个是H5没有出来之前使用的方法。
使用两个input框，一个text，一个password，初始状态，password隐藏；实质就是两个input之间的切换。

    <input id="showPwd" class="txt" type="text" value="请输入密码" tabindex="2" />
    <input id="pwd" class="txt" name="password" type="password" style="display:none;"/>
        
    <script type="text/javascript">
    $(function(){
        var showPwd = $("#showPwd"), pwd = $("#pwd");
        showPwd.focus(function(){
            pwd.show().focus();
            showPwd.hide();
        });

        pwd.blur(function(){
            if(pwd.val()=="") {
                showPwd.show();
                pwd.hide();
            }
        });
    });
    </script>

2.HTML5出现后，就简单多了，现在只需要设置placeholder就可以了

    <input type="password" placeholder="请输入密码"/>

placeholder 属性提供可描述输入字段预期值的提示信息（hint）；
适用于以下的 <input> 类型：text, search, url, telephone, email 以及 password。
